# Виртуальный тур по галактике

 Проект создан при поддержке 13 кружек воды и двух шоколадок. Проект сделан с душой и работой множества людей, 
которые меня поддерживали и помогали с проектом, больше всего была проблема с поиском интерактивной карты и 
настройкой координат, но я справился. Желающие могут посмотреть мой проект и предложить решения по улучшению 
или заимствования для себя. Дальнейшие улучшения проекта было приостановлено, так как его функционал был рассчитан 
для вуза.

##  Функциональность

*   **Интерактивная карта:** Поиск и навигация по объектам.
*   **Туры:** создования и просмотр туры других ползователей
*   **Отзевы:** отправка и коментирования туров других

##  Установка и запуск

# 1. Клонируйте репозиторий

# 2. запустите его через Визуал Студио
